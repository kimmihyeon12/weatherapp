# 날씨 앱

날씨 api를 사용한 날씨 어플케이션

# 어플 기능

- 현재 위치 표기
- 현재 날씨 및 오늘의 최고/최저기온 표기
- 시간대별 날씨 표기(48시간)
- 일주일간 최저 / 최고기온 표기
- 날씨별 배경 및 아이콘 바뀌도록 함

# 학습내용

- Geolocator 학습
- http get 메서드 사용법 및 response class(resonse.body , response.statuscode) 학습
- json parsing (jsonDecode) 학습    
- getx를 통한 로직 분리
